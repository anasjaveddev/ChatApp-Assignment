# 💬 Chat App Assignment

A simple chat application built using **HTML, CSS, and JavaScript**.
This project allows users to type messages and display them instantly on the screen.

---

## 📌 Features

* User can type a message
* Messages are displayed instantly in the chat area
* Input field clears after sending a message
* Simple and beginner-friendly UI
* Uses basic DOM manipulation

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

1. User types a message in the input field
2. Clicks the **Send** button
3. Message is stored in an array
4. A new `<p>` element is created
5. Message is displayed inside the chat box

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
